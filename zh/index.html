GriefDefender

一个为 Bukkit 及 mods 服务器设计的领地插件!

如何使用 💡 Github 源码 (opens new window)

性能优异 🚀

没有什么是被硬编码的。高效，轻量，流畅。

高自定义 🧰

使用默认的权限配置或根据标签创建属于您自己的，可能无上限!

多语言支持 🌏

许多语言开箱即用，不需要额外的配置!

多种数据库支持 📇

支持 MySQL，MariaDB，PostgreSQL 及 SQLite(文件型)。

交互式 UI 📱

通过点击文本来处理插件事物。

Mod 支持 🛠

久经 Mod 考验，通过 GD事件 来创建标签来保护您的服务器。支持 混合 & Sponge 端。

税收，出租，及出售💰

通过出租，出售等经济行为来为您的服务器经济添砖加瓦。

广泛的插件支持 🛡

通过安装 GDHooks 谋取与其它插件的兼容。

自动恢复及结构图 🔄

通过世界生成或结构图复原领地。

权限数据节点化

通过权限处理一切事物。LuckPerms 是必要前置。

精心设计的 API

通过 API 挂钩 GriefDefender 的所有行为。

从别的插件迁移

目前支持 WorldGuard 和 GriefPrevention 的领地迁移。

还有更多待你发掘 ✨

请将您的想法及时告诉我们，我们一切尽最大的努力将 GriefDefender 变得更好。
